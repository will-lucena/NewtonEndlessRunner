* Implementado movimento com touch

  * Clicar no lado direito da tela -> move pra direita
  * Slide pra direita -> move pra direita
  * Slide pra esquerda -> move pra esquerda
  * Slide pra cima -> pula
  * Slide pra baixo -> slide
